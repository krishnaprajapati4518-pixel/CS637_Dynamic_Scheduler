# CS637 Dynamic Scheduler

This repository implements a dynamic scheduling framework for Cyber-Physical Systems (CPS)
based on the EMSOFT24 paper "Revisiting Dynamic Scheduling of Control Tasks: A Performance-Aware Fine-Grained Approach".

## Features
- LTI System Modeling
- LMI-based Stability Analysis
- Control Skipping Automaton (CSA) Synthesis
- Online Dynamic Scheduler
- Energy-Aware Co-Design
- GRU-based ML Predictor

## Usage
```
python src/system_model.py
```
